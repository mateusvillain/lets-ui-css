# Color

Tokens de cor do sistema de design. As cores estão organizadas em três camadas: **primitivas** (escalas brutas), **marca** (paleta da marca) e **semânticas** (papéis de uso).

---

## Cores Primitivas

Escalas de 8 tons, do mais claro ao mais escuro. Use a função SCSS `color($palette, $scale)` para acessá-las.

{primitives.map(({ name, key, shades }) => (
  <div key={key} style={{ marginBottom: '32px' }}>
    <p style={{ margin: '0 0 8px', fontWeight: 500, fontSize: '14px' }}>{name}</p>
    <div style={{ display: 'grid', gridTemplateColumns: 'repeat(8, 1fr)', gap: '6px' }}>
      {shades.map((hex, i) => (
        <div key={i}>
          <div style={{
            height: '48px',
            background: `var(--lui-color-${key}-${i + 1})`,
            borderRadius: '6px',
            border: '1px solid rgba(0,0,0,0.08)',
          }} />
          <code style={{ display: 'block', fontSize: '10px', marginTop: '4px', lineHeight: 1.3 }}>
            {`--lui-color-${key}-${i + 1}`}
          </code>
          <span style={{ display: 'block', fontSize: '10px', color: 'var(--lui-color-neutral-text-caption)' }}>
            {hex}
          </span>
        </div>
      ))}
    </div>
  </div>
))}

### Black Alpha

Sobreposições em preto com opacidade crescente, usadas em estados de hover e pressed.

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(8, 1fr)', gap: '6px', marginBottom: '32px' }}>
  {[
    ['1','2%'],['2','4%'],['3','8%'],['4','12%'],
    ['5','16%'],['6','20%'],['7','24%'],['8','28%'],
  ].map(([n, alpha]) => (
    <div key={n}>
      <div style={{
        height: '48px',
        background: `var(--lui-color-black-alpha-${n})`,
        borderRadius: '6px',
        border: '1px solid rgba(0,0,0,0.08)',
        backgroundColor: `rgba(0,0,0,${parseFloat(alpha)/100})`,
      }} />
      <code style={{ display: 'block', fontSize: '10px', marginTop: '4px', lineHeight: 1.3 }}>
        {`--lui-color-black-alpha-${n}`}
      </code>
      <span style={{ display: 'block', fontSize: '10px', color: 'var(--lui-color-neutral-text-caption)' }}>
        {alpha} opacidade
      </span>
    </div>
  ))}
</div>

## Cores da Marca

As cores primária e secundária são definidas por arquivo de tema e podem variar por marca (`data-brand`). O tema padrão usa violeta como primária e cinza como secundária.

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(2, 1fr)', gap: '24px', marginBottom: '32px' }}>
  <div>
    <p style={{ margin: '0 0 8px', fontWeight: 500, fontSize: '14px' }}>Primary</p>
    <div style={{ display: 'grid', gridTemplateColumns: 'repeat(8, 1fr)', gap: '4px' }}>
      {[1,2,3,4,5,6,7,8].map(n => (
        <div key={n}>
          <div style={{
            height: '40px',
            background: `var(--lui-brand-color-primary-${n})`,
            borderRadius: '4px',
            border: '1px solid rgba(0,0,0,0.06)',
          }} />
          <code style={{ display: 'block', fontSize: '9px', marginTop: '3px' }}>{n}</code>
        </div>
      ))}
    </div>
  </div>
  <div>
    <p style={{ margin: '0 0 8px', fontWeight: 500, fontSize: '14px' }}>Secondary</p>
    <div style={{ display: 'grid', gridTemplateColumns: 'repeat(8, 1fr)', gap: '4px' }}>
      {[1,2,3,4,5,6,7,8].map(n => (
        <div key={n}>
          <div style={{
            height: '40px',
            background: `var(--lui-brand-color-secondary-${n})`,
            borderRadius: '4px',
            border: '1px solid rgba(0,0,0,0.06)',
          }} />
          <code style={{ display: 'block', fontSize: '9px', marginTop: '3px' }}>{n}</code>
        </div>
      ))}
    </div>
  </div>
</div>

## Cores Semânticas

Cores com papel de uso definido. Use as funções SCSS `bg()`, `text()`, `stroke()`, `icon()` para acessá-las nos componentes.

{semanticGroups.map(({ label, prefix, tokens }) => (
  <div key={prefix} style={{ marginBottom: '32px' }}>
    <p style={{ margin: '0 0 10px', fontWeight: 500, fontSize: '14px' }}>{label}</p>
    <div style={{ display: 'flex', flexDirection: 'column', gap: '4px' }}>
      {tokens.map(({ role, desc }) => (
        <div key={role} style={{
          display: 'grid',
          gridTemplateColumns: '32px 1fr auto',
          alignItems: 'center',
          gap: '12px',
          padding: '8px 12px',
          background: 'var(--lui-color-neutral-background-surface)',
          border: '1px solid var(--lui-color-neutral-border-subtle)',
          borderRadius: '6px',
        }}>
          <div style={{
            width: '32px',
            height: '32px',
            borderRadius: '4px',
            background: `var(--lui-color-${prefix}-${role})`,
            border: '1px solid rgba(0,0,0,0.08)',
            flexShrink: 0,
          }} />
          <code style={{ fontSize: '12px' }}>{`--lui-color-${prefix}-${role}`}</code>
          <span style={{ fontSize: '11px', color: 'var(--lui-color-neutral-text-caption)', textAlign: 'right' }}>{desc}</span>
        </div>
      ))}
    </div>
  </div>
))}

## Link

Cores para elementos de link. Herdam da cor primária da marca e mudam por tema.

<div style={{ display: 'flex', flexDirection: 'column', gap: '4px', marginBottom: '16px' }}>
  {[
    { token: '--lui-color-link-default', desc: 'Link padrão'   },
    { token: '--lui-color-link-visited', desc: 'Link visitado' },
  ].map(({ token, desc }) => (
    <div key={token} style={{
      display: 'grid',
      gridTemplateColumns: '32px 1fr auto',
      alignItems: 'center',
      gap: '12px',
      padding: '8px 12px',
      background: 'var(--lui-color-neutral-background-surface)',
      border: '1px solid var(--lui-color-neutral-border-subtle)',
      borderRadius: '6px',
    }}>
      <div style={{
        width: '32px',
        height: '32px',
        borderRadius: '4px',
        background: `var(${token})`,
        border: '1px solid rgba(0,0,0,0.08)',
        flexShrink: 0,
      }} />
      <code style={{ fontSize: '12px' }}>{token}</code>
      <span style={{ fontSize: '11px', color: 'var(--lui-color-neutral-text-caption)', textAlign: 'right' }}>{desc}</span>
    </div>
  ))}
</div>
