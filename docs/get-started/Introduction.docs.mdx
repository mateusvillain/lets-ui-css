# Introduction

**Let's UI** é um design system open-source, framework-agnostic, construído sobre CSS custom properties e Web Components nativos. Funciona em qualquer projeto — React, Vue, Angular, Svelte ou HTML puro — sem impor dependências de runtime.

## Instalação

Escolha a forma de instalação de acordo com o que o seu projeto precisa.

### Somente CSS

A maneira mais simples de usar o Let's UI. Instale o pacote de estilos — ele inclui o CSS compilado com todos os componentes, foundations e reset.

```bash
npm install @lets-ui/styles
```

Importe o CSS no ponto de entrada da sua aplicação:

```js
import '@lets-ui/styles';
```

Ou via `<link>` no HTML:

```html
<link rel="stylesheet" href="node_modules/@lets-ui/styles/dist/letsui.min.css" />
```

---

### Com Web Components

Para usar os componentes interativos (botões, inputs, modais, etc.), importe o pacote de componentes inteiro ou componentes individuais. Os estilos são injetados automaticamente — nenhum import de CSS adicional é necessário.

```bash
npm install @lets-ui/components
```

```js
// Registra todos os componentes
import '@lets-ui/components';

// Ou importe componentes individualmente
import '@lets-ui/components/button';
import '@lets-ui/components/modal';
```

Os componentes se registram automaticamente como custom elements e ficam disponíveis no HTML:

```html
<lui-button variant="primary">Confirmar</lui-button>
<lui-button variant="secondary">Cancelar</lui-button>
```

---

### Com Tokens SCSS

Para projetos que precisam dos tokens de design nas suas próprias folhas de estilo SCSS (usando as funções `bg()`, `text()`, `radius()`, etc.):

```bash
npm install @lets-ui/tokens utilities
```

Configure o `@use` no topo dos seus arquivos SCSS (requer `--load-path=node_modules` no Sass):

```scss
@use '@lets-ui/tokens/dist/letsui.tokens.scss' as *;
@use 'utilities/src/functions' as *;

.my-component {
  background: bg(container, primary);
  color: text(body);
  border-radius: radius(md);
}
```

## Temas e marcas

O Let's UI suporta modo claro/escuro e múltiplas marcas via atributos HTML, sem JavaScript adicional.

### Modo claro / escuro

Aplique `data-theme` no elemento raiz. O valor padrão é `light`.

```html
<!-- Modo claro (padrão) -->
<html data-theme="light">

<!-- Modo escuro -->
<html data-theme="dark">
```

Para alternar o tema via JavaScript:

```js
document.documentElement.setAttribute('data-theme', 'dark');
```

### Multi-marca

O atributo `data-brand` define qual paleta de marca é usada. O tema padrão é `lets-ui`.

```html
<html data-theme="light" data-brand="lets-ui">
```

Tokens de marca como `--lui-brand-color-primary-*` e `--lui-brand-border-radius-*` são sobrescritos automaticamente ao trocar o `data-brand`.
