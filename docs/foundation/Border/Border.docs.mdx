# Border

Tokens de borda do sistema de design. Controlam o arredondamento e a espessura das bordas — use as funções SCSS `radius($size)` e `width($size)` para acessá-los nos componentes.

---

## Border Radius

Escala de arredondamento, do reto ao completamente circular. Aplicado via `radius($size)`.

<div style={{ display: 'flex', flexDirection: 'column', gap: '4px', marginBottom: '32px' }}>

      gap: '16px',
