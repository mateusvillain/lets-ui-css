# Typography

Sistema tipográfico do design system. As escalas compostas combinam família, tamanho, peso e capitalização — use a função SCSS `font-scale($name)` ou a classe CSS `.{name}`.

---

## Escalas Compostas

Cada escala define um papel tipográfico completo. A classe CSS correspondente pode ser aplicada diretamente no HTML.

<div style={{ display: 'flex', flexDirection: 'column', gap: '2px', marginBottom: '32px' }}>
  {scales.map(({ name, className, weight, size, use }) => (
    <div key={name} style={{
      display: 'grid',
      gridTemplateColumns: '140px 1fr 200px',
      alignItems: 'center',
      gap: '16px',
      padding: '16px',
      border: '1px solid var(--lui-color-neutral-border-subtle)',
      borderRadius: '6px',
    }}>
      <div>
        <code style={{ fontSize: '12px', display: 'block' }}>.{name}</code>
        <span style={{ fontSize: '11px', color: 'var(--lui-color-neutral-text-caption)' }}>
          {weight} · {size}
        </span>
      </div>
      <div className={className} style={{ overflow: 'hidden', textOverflow: 'ellipsis', whiteSpace: 'nowrap' }}>
        O design system cuida dos detalhes.
      </div>
      <span style={{ fontSize: '11px', color: 'var(--lui-color-neutral-text-caption)', textAlign: 'right' }}>
        {use}
      </span>
    </div>
  ))}
</div>

## Tamanhos de Fonte

Escala fluida com `clamp()` — os tamanhos se adaptam proporcionalmente entre os breakpoints mínimo e máximo.

<div style={{ display: 'flex', flexDirection: 'column', gap: '4px', marginBottom: '32px' }}>
  {fontSizes.map(({ token, var: cssVar, min, max }) => (
    <div key={token} style={{
      display: 'grid',
      gridTemplateColumns: '60px 260px 1fr',
      alignItems: 'center',
      gap: '16px',
      padding: '8px 12px',
      background: 'var(--lui-color-neutral-background-surface)',
      border: '1px solid var(--lui-color-neutral-border-subtle)',
      borderRadius: '6px',
    }}>
      <code style={{ fontSize: '12px' }}>{token}</code>
      <code style={{ fontSize: '11px', color: 'var(--lui-color-neutral-text-caption)' }}>{cssVar}</code>
      <span style={{ fontSize: `var(${cssVar})`, lineHeight: 1, overflow: 'hidden' }}>Aa</span>
    </div>
  ))}
</div>
