# Spacing

Tokens de espaçamento do sistema de design. Existem duas escalas: **fixo** (valores absolutos em px) e **fluido** (valores responsivos com `clamp()` que se adaptam entre breakpoints). Use as funções SCSS `fixed($size)` e `fluid($size)` para acessá-los nos componentes.

---

## Espaçamento Fixo

Valores absolutos em pixels — ideais para espaçamentos internos de componentes que não devem mudar com o viewport.

<div style={{ display: 'flex', flexDirection: 'column', gap: '4px', marginBottom: '32px' }}>
  {fixedTokens.map(({ name, token, value, px }) => (
    <div key={name} style={{
      display: 'grid',
      gridTemplateColumns: '200px 80px 1fr',
      alignItems: 'center',
      gap: '16px',
      padding: '10px 16px',
      background: 'var(--lui-color-neutral-background-surface)',
      border: '1px solid var(--lui-color-neutral-border-subtle)',
      borderRadius: '6px',
    }}>
      <div>
        <code style={{ display: 'block', fontSize: '12px' }}>{token}</code>
        <span style={{ fontSize: '11px', color: 'var(--lui-color-neutral-text-caption)' }}>
          fixed({name})
        </span>
      </div>
      <span style={{ fontSize: '12px', color: 'var(--lui-color-neutral-text-body)', fontVariantNumeric: 'tabular-nums' }}>{value}</span>
      <div style={{ display: 'flex', alignItems: 'center' }}>
        <div style={{
          height: '12px',
          width: `${Math.max(px, 2)}px`,
          background: 'var(--lui-color-primary-background-container)',

          minWidth: px === 0 ? '2px' : undefined,
          opacity: px === 0 ? 0.3 : 1,
        }} />
      </div>
    </div>
  ))}
</div>

## Espaçamento Fluido

Valores responsivos com `clamp()` — escalam proporcionalmente entre o breakpoint mínimo (320 px) e máximo (1440 px). Ideais para padding de seções, gaps de layout e margens entre blocos.

<div style={{ display: 'flex', flexDirection: 'column', gap: '4px', marginBottom: '16px' }}>
  {fluidTokens.map(({ name, token, clamp, min, max }) => (
    <div key={name} style={{
      display: 'grid',
      gridTemplateColumns: '200px 120px 80px 1fr',
      alignItems: 'center',
      gap: '16px',
      padding: '10px 16px',
      background: 'var(--lui-color-neutral-background-surface)',
      border: '1px solid var(--lui-color-neutral-border-subtle)',
      borderRadius: '6px',
    }}>
      <div>
        <code style={{ display: 'block', fontSize: '12px' }}>{token}</code>
        <span style={{ fontSize: '11px', color: 'var(--lui-color-neutral-text-caption)' }}>
          fluid({name})
        </span>
      </div>
      <span style={{ fontSize: '10px', color: 'var(--lui-color-neutral-text-caption)', fontVariantNumeric: 'tabular-nums' }}>
        {min} → {max}

      </div>
    </div>

</div>
